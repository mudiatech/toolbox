# Gearbox

Common utilities


#### How to Use
1. Add maven repository

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

2. Add this project as maven dependency

```xml
<dependency>
    <groupId>com.gitlab.muhammadkholidb</groupId>
    <artifactId>gearbox</artifactId>
    <version>master-SNAPSHOT</version>
</dependency>
```
